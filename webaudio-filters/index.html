Web audio: Filter Playground

On this page you can experiment with the filters provided by the Web Audio specifications. To run this you need a current build of chrome, firefox or safatir to work. If you want to allow microphone input, make sure you use headphones for output. For more information on how to get started with Web Audio and how this works, look at this article on http://www.smartjava.org or contact me on twitter:
